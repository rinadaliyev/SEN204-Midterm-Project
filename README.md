# IGU-Brandcolors

IGU-BrandColors is a web-based application developed as a **midterm project** for the Internet and Web Programming course. It helps designers and developers easily find and use official brand color palettes.

## 🎯 Project Purpose

The project aims to simplify access to accurate brand colors, improving efficiency and consistency in design workflows. With a clean and user-friendly interface, users can quickly search for brands and copy their color codes.

---

## ✨ Features

- 🔍 Search functionality for different brands  
- 🎨 Display of official brand color palettes  
- 📋 Easy copying of HEX/RGB color values  
- 💻 Simple, friendly user interface  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript
